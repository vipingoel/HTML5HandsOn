Input controls

Demo for input controls is shown in the header of this application.

  1. On top right side, there is a search box shows place holder.
  2. After search box, there is a color input box. This will be best viewed on Opera browser. Set any color to change theme

Navigator.Online

Used navigator.online property to show status of connection. Whether online or offline. Works with google chrome but not working on other browsed when system goes offline. Chrome correcly updates the status from online to offline.

Geolocation API

Used geolocation API to fetch current location and mapped it using google maps API.

Local storage and Session storage

  • Selected Theme color is saved in localStorage so it will persist if we reopen browser.
  • Current location is saved in session storage and will persist if we browse any other form
